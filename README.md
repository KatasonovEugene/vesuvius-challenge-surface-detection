# Vesuvius Challenge: Surface Detection

This repository contains our solution for the Kaggle competition [Vesuvius Challenge - Surface Detection](https://www.kaggle.com/competitions/vesuvius-challenge-surface-detection)

The task is to detect the surface of papyrus scrolls using 3D CT scan data

# 🥉 Results
- Bronze medal for top 10% solution

- Ranked among 1391 teams
